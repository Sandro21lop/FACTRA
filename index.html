El comprobante estándar para cualquier tipo de negocio. Se imprime en impresora térmica al instante y al mismo tiempo envía la factura electrónica al SRI y al correo del cliente.

✓ Tienda / Minimarket ✓ Venta + Servicios ✓ Solo Servicios ✓ Barbería / Salón ✕ No aplica en Restaurante

  • Número de factura secuencial (001-001-000000001)
  • Detalle de productos, cantidades y precios
  • Subtotal, IVA, descuentos y total
  • Forma de pago y cambio
  • QR con código de autorización SRI
  • Mensaje de código de descuento para referidos

Ticket de Restaurante + Comanda de Cocina

Diseñado para el flujo de un restaurante: al cobrar se imprime el ticket de venta para el cliente y automáticamente una comanda para cocina con el número de pedido y las notas del plato.

✓ Restaurante ✓ Cafetería ✓ Comida rápida ✕ No aplica en tienda o servicios
  • Ticket estándar de venta con facturación electrónica incluida
  • Número de pedido grande y visible en ambos tickets
  • Comanda automática para cocina al mismo tiempo
  • Notas por plato (ej: "Sin cebollas", "Término medio")
  • Hora del pedido y referencia de factura en la comanda
  • QR de autorización SRI en el ticket del cliente

Facturación SRI

Sin trámites,
sin complicaciones

Factra Ecuador maneja todo el proceso de facturación electrónica. Desde generar el XML hasta enviar la factura al correo del cliente.

  • Factura electrónica generada y autorizada por el SRI automáticamente
  • XML autorizado descargable por cada venta
  • PDF con QR code y clave de acceso para el cliente
  • La factura llega al correo del cliente automáticamente
  • ATS mensual generado en XML listo para presentar al SRI
  • Notas de crédito, débito y liquidaciones incluidas
  • Funciona en ambiente de pruebas y producción SRI
